# Icons

Place icon files here:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)
- icon16-recording.png (16x16, red/recording state)
- icon48-recording.png (48x48, red/recording state)
- icon128-recording.png (128x128, red/recording state)

You can use any PNG icons. The extension will work without icons (browser will show a default icon), but it's recommended to add proper icons for a polished look.

Quick icon generation: Use https://favicon.io/ or any icon generator to create a simple "network" or "save" themed icon set.
